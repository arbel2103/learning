# מרחב הלמידה 📚

אפליקציית לימודים אישית — מחברת דיגיטלית מובנית. ממשק בעברית (RTL), אפליקציית צד-לקוח בלבד (SPA) ללא שרת וללא חשבונות. כל הנתונים נשמרים מקומית בדפדפן.

🔗 **אתר חי:** https://arbel2103.github.io/learning/

## יכולות

- **פרויקטים** בסרגל עליון — הוספה, שינוי שם, ומחיקה (עם אישור).
- **נושאים (טאבים)** בתוך כל פרויקט — בריחוף עכבר נפתחת רשימת תתי-הנושאים.
- **תתי-נושאים** — לכל אחד מרחב למידה משלו.
- **מרחב למידה (מחברת שורות):**
  - כתיבת סיכומים על שורות, עם מספור שורות (כמו Word) ורשימות (• תבליט / מספור).
  - הוספת **תמונות עזר**, **קבצי עזר** וקישורי **סרטוני YouTube**.
  - בחירת שורות והפיכתן ל**אזור ציור חלק** — לציור גרפים וכו'.
  - **סרגל ציור** עם עט, בורר צבע, מד עובי, ומחק (כמו לוח מחיק).
  - החזרת שורות והזזת בלוקים למעלה/למטה.
- **גיבוי** — ייצוא/ייבוא של כל הנתונים (כולל תמונות) לקובץ JSON.

## טכנולוגיות

React + Vite + TypeScript + Tailwind CSS · Zustand (persist/localStorage) · IndexedDB (קבצים).

## פיתוח

```bash
npm install
npm run dev      # שרת פיתוח
npm run build    # בנייה לייצור
```

הפריסה אוטומטית ל-GitHub Pages בכל push ל-`main` (ראו `.github/workflows/deploy.yml`).
